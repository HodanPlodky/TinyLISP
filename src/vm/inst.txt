LDC x - push x on stack
NIL - push nil on stack
SEL x y rest - if top of stack /= 0 -> conti on x else conti y dump rest
JOIN - pop from dump
ADD SUB MUL DIV CONS - binary op
CAR CDR CONSP - unary op
LD - loads argument from env and pushes it on stack
LDF - takes the code and the current env and pushes closure on stack
AP - apply args to env
RTN - return last val
